# AI-Powered Sleep Apnea Prediction and Healthcare Recommendation System

## Overview

This project is a Flask-based web application that predicts Sleep Apnea severity using Machine Learning models and provides personalized healthcare recommendations.

The system analyzes patient health parameters such as age, gender, weight, height, blood pressure, pulse rate, Oxygen Desaturation Index (ODI), Apnea Index (AI), and Hypopnea Index (HI) to estimate the Apnea-Hypopnea Index (AHI).

## Features

* Sleep Apnea Severity Prediction
* BMI Calculation
* Sleep Quality Assessment
* Personalized Healthcare Recommendations
* User-Friendly Web Interface
* Machine Learning-Based Risk Analysis

## Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* HTML
* CSS
* JavaScript

## Project Workflow

1. User enters health parameters.
2. System calculates BMI.
3. Machine Learning model predicts AHI.
4. Sleep quality is classified.
5. Personalized recommendations are generated.
6. Results are displayed through the web interface.
